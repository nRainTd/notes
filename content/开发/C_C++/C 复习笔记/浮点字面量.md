## 1 说明  

浮点字面量，课本里叫实型常量，实际上就是能在 c 源码中直接写出来的表示小数的量。  
例如 `0.1` `2e6` 等。

## 2 两种形式  

浮点字面量有两种表示形式：小数形式和指数形式  

### 2.1 小数形式  

1. 由符号（-）、数字和小数点组成
2. 必须要有小数点，但小数点的位置是**任意的** 

例如：  

```
0.0 0.345 2.0 1.23 .25 300. -123.456
```

都是合法的浮点数  
其中，`.25` 其实是 `0.25` ；`300.` 其实是 `300.0`  

### 2.2 指数形式  

一般形式：  

``` c
<a>e<n> 或 <a>E<n>
```

其中，
1. `<a>` 为基数， 分正负，可以没有小数点
2. `<n>` 为阶码， 整数，分正负  
其值为：  

$$
a \times 10 ^ n
$$

注意：  
1. 基数和阶码都要有  

### 2.3 提示  

一般浮点字面量的数据类型为 `double` ，如果想要表示 `float` 类型的字面量，可以在浮点字面量后面加上 `f` 。例如 `123.456f` 表示这是一个 `float` 类型的字面量
