## 1 说明  

整型字面量，课本上叫整型常量，其实就是能直接在源码中书写的用来表示整型的字面意义的量  

## 2 表示形式  

整型字面量有八进制、十进制、十六进制三种表示方法  

> [!tip] 注意
> c 语言的整型不能直接表示为二进制的形式  

### 2.1 八进制形式  

以 0 开头，后面各位只能取 0 ~ 7  

### 2.2 十进制形式  

首位不能为 0，后面各位取 0 ~ 9  

### 2.3 十六进制形式  

以 数字 0 加上字母 x(X) 开头，后面各位取 0 ~ 9 $\bigcup$ a ~ f || A ~ F  

### 2.4 注意  

八进制和十六进制形式都只能表示非负整数  

## 3 提示  

整型字面量的类型  
1. 如果没有任何后缀，  
	1. 该整数在有符号普通整型范围内，就是此类型  
	2. 该整数在无符号普通整型范围内，就是此类型  
2. 有后缀 U(u)，
	无符号普通整型  
3. 有后缀 L(l)，
	有符号长整型   
4. 同时有后缀 U(u) L(l)，
	无符号长整型
