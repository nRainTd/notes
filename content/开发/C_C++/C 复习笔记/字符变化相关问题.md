# 关键字符的 ASCII 值  

1. '0' ~ '9' : 48 ~ 57  
2. 'A' ~ 'Z' : 65 ~ 90  
3. 'a' ~ 'z' :  97 ~ 122  

# 字母大小写转换问题  

假设有字符 ch:   
分两种情况，每种情况有两种方法  
1. 若 ch 为小写字母，要把它变成大写字母
	1. 减去 ('a' - 'A')，即减去 32
	2. 用 'A' + (ch - 'a')  
2. 若 ch 为大写字母，要把它变成小写字母  
	1. 加上 ('a' - 'A')，即加上 32
	2. 用 'a' + (ch - 'A') 

对于第一种方法，需要知道大写字母的 ASCII 值小于小写字母  

# 字符移位问题  

## 1 字母移位  

由于大小写字母的处理方法一致，这里只以小写字母为例    

需要知道字母一共有 26 个  

对于给定的字符 ch 和给定的位移 d:    
首先对 d 进行 d %= 26;  
分两个移动方向，每个移动方向分两种小情况    
1. 向后移动  
	1. 如果 ch 在 'a' ~ ('z' - d) 范围内
		ch + d
	2. 如果 ch 在 ('z' - d + 1) ~ 'z' 范围内  
		ch - (26 - d)
2. 向前移动  
	1. 如果 ch 在 ('a' + d) ~ 'z' 范围内  
		ch - d
	2. 如果 ch 在 'a' ~('a' + d - 1) 范围内
		ch + (26 - d)

## 2 数字移位  

前提是知道数字从 0 开始，以 9 结束，共 10 个  

对于给定数字字符 ch 和给定位移 
首先对 d 进行 d %= 10;  
1. 向后移动  
	1. 如果 ch 在 '0' ~ ('9' - d) 范围内
		ch + d
	2. 如果 ch 在 ('9' - d + 1) ~ '9' 范围内  
		ch - (10 - d)
2. 向前移动  
	1. 如果 ch 在 ('0' + d) ~ '9' 范围内  
		ch - d
	2. 如果 ch 在 '0' ~('0' + d - 1) 范围内
		ch + (10 - d)
