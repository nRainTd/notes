## 1 常见转义字符

| 转义字符   | 描述         | ASCII 值（十进制） |
| ------ | ---------- | ------------ |
| `\n`   | 换行符        | 10           |
| `\t`   | 水平制表符（Tab） | 9            |
| `\r`   | 回车符        | 13           |
| `\b`   | 退格符        | 8            |
| `\f`   | 换页符        | 12           |
| `\\`   | 反斜杠        | 92           |
| `\'`   | 单引号        | 39           |
| `\"`   | 双引号        | 34           |
| `\?`   | 问号         | 63           |
| `\0`   | 空字符（null）  | 0            |
| `\v`   | 垂直制表符      | 11           |
| `\xhh` | 表示十六进制的字符值 |              |
| `\ooo` | 表示八进制的字符值  |              |

## 2 特殊情况  

### 2.1 \xhh  

1. 说明：用来表示 ASCII 码值为 十六进制 hh 的字符  
2. 规则： 可以是 \xh 也可以是 \xhh  

### 2.2 \ooo  

1. 说明：用来表示 ASCII 码值为 八进制 ooo 的字符  
2. 规则：
	1. o 只能为 0~7  
	2. 可以为 \ooo 、\oo 或 \o  

### 2.3 \字符或特定符号

1. 如果有特定含义，如 \n \r，则为代表其对应的特定字符  
2. 如没有特定含义，如 \z `\}`，则代表原来的字符（如 `\z == z` 、`\} == }`
3. \x 是非法字符
