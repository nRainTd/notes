# 十进制转其他进制  

## 1 整数部分  

将十进制整数部分 d 转为 n 进制数   
设有一个数组 r\[]  
1. r.push(d % n)
2. d /= n，如果 d != 0 重复执行步骤 1. 2. ，直到 d == 0 为止  
3. r.pop() 从高位开始不断放，直到数组 r\[] 空了为止  
4. 得到的数即为十进制数 d 转为 n 进制后的数  

例如，将十进制数 11 转为二进制数  
![[Drawing 2025-01-03 20.33.01.excalidraw.png |250]]  
结果为：1011  

## 2 小数部分  

将十进制小数部分 d 转为 n 进制数   
设一个数组 z\[]  
1. $d \times 2$ 取整数部分 push 进 z\[]  
2. 将 d 整数部分变为 0，如果 d != 0.0 重复执行步骤 1. 2. ，直到 d == 0.0 为止  
3. r.shift() 从小数点后开始不断放，直到数组 z\[] 空了为止

例如，将十进制数 0.125 转为二进制数  
![[Drawing 2025-01-03 21.22.14.excalidraw.png |250]]  
结果为： 0.001  

# 其他进制数转为十进制数  

将 n 进制数 x 转为十进制数  
假设特定位的数 $a_i$，其中，下标 $i$ 代表位权  
十进制数 = $\sum(a_i \times n^i)$    

例如，将二进制数 1011.001 转为十进制数  

$$
\begin{align}
[1011.001]_{二进制} &= 
[1 \times 2^{3} + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0 + 0 \times 2^{-1} + 0 \times 2^{-2} + 1 \times 2^{-3}]_{十进制} \\
&= [11.125]_{十进制}
\end{align}
$$

# 二进制、八进制、十六进制互相转换  

## 1 八进制、十六进制转换为二进制  

先把每一位单独转换为对应的二进制，然后直接按照顺序拼在一起即可  

例如，八进制数 123 转换为二进制：  
$[1]_{八进制} = [001]_{二进制}$
$[2]_{八进制} = [010]_{二进制}$
$[3]_{八进制} = [011]_{二进制}$
所以：$[123]_{八进制} = [001,010,011]_{二进制}$

## 2 二进制转换为八进制、十六进制  

 
1. 如果转换为八进制，则从最低位开始三个为一组；如果转换为十六进制，则从最低位开始四个为一组  
2. 分好组后，在把每组转换为对应的八进制、十六进制  
3. 最后直接按照顺序拼接即可  

例如，二进制数 1010011 转换为十六进制  
1. 分组：$[0101,0011]_{二进制}$   
2. 转换：
	$[0101]_{二进制} = [5]_{十六进制}$ 
	$[0011]_{二进制} = [3]_{十六进制}$ 
3. 所以：$[0101,0011]_{二进制} = [53]_{十六进制}$  

## 3 八进制、十六进制互相转换  

分为两步：  
1. 先使用上述标题 1 的方法将八进制（十六进制）转换为二进制  
2. 再使用上述标题 2 的方法将二进制转换为十六进制（八进制）  

例如，八进制 123 转换为十六进制  
$[123]_{八进制} = [001,010,011]_{二进制} = [0101,0011]_{二进制} = [53]_{十六进制}$
